# 100-mexicanos-dijeron

A "Family Feud" style trivia game, inspired by the famous Mexican TV show "100 Mexicanos Dijeron". 

The game presents a prompt and requires the player to guess the most popular answers among 100 surveyed people. Points are awarded based on the survey response value.

**Tech Stack:** HTML, CSS, JavaScript.
**Key Features:** Survey-based gameplay, simple and intuitive UI, and score tracking.
